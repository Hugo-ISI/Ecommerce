Web programming course

CSS (Cascading Style Sheets) is a language used to describe the appearance of a page. It allows you to target different HTML elements to then describe their presentation.

As we saw in the previous training, HTML is a language that allows you to define the structure of a document. CSS, for cascade style sheet, is a language that will allow us to define the form to be given to our document. It has a simple syntax but a fairly substantial vocabulary that you will have to learn and master while practicing. In this training, we will start by discovering language and how it works before delving more deeply into the properties to know. We will then end with a practical case that will allow you to discover a concrete case of use of CSS and to solidify your knowledge.
